# 🏗️ Current Project — ShareStay
> **지역 안전도를 반영한 방 공유·매물 플랫폼**  
> Spring Boot + React 완전 분리형 구조로 직접 개발 중

### 🔹 핵심 기능
- JWT 기반 인증 + OAuth2(Google Login)
- 매물 등록 / 조회 / 검색
- 지역 기반 필터링 및 정렬
- React 컴포넌트 모듈화 + React Query
- Spring Security 6.x 기반 인증/인가 구조
- MariaDB를 활용한 관계형 스키마 설계

### 🔹 현재 집중하고 있는 부분
- Spring Security의 **정교한 인증 흐름 리팩터링**
- API 안정성 향상 및 예외 구조 정리
- React 컴포넌트 구조 재정비
- DB 정규화 및 스키마 최적화
- 매물 추천 알고리즘 설계 초안
